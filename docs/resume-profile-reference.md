    - 이를 통해 캠페인 실행 리드타임을 단축하고, 마케팅 유입 데이터의 정확한 트래킹 환경을 마련하여 전사적 협업 효율을 높였습니다.

    - 디자인 시스템 도입을 통해 신규 화면 개발 속도를 가속화하고, 서비스 전반의 UI 일관성을 확보하여 팀 전체의 협업 효율 및 생산성을 높였습니다.
